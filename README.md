# 🚀 ISRO Mission Data Analysis Dashboard

## 📊 Overview
This project analyzes ISRO mission data using SQL and Power BI to uncover insights related to mission success rate, cost, mission types, and yearly trends. The dashboard provides an interactive and visual representation of space mission data.

## 🛠 Tech Stack
- Power BI
- SQL
- DAX
- CSV Datasets

## 📈 Dashboard Features
- Vehicle-wise Success Rate Analysis
- Year-wise Mission Trends
- Mission Cost Comparison (Top N filtering)
- Mission Type Distribution
- Interactive slicers for dynamic filtering

## 🔍 Key Insights
- PSLV has the highest mission success rate
- Chandrayaan-2 is among the highest-cost missions
- Mission frequency has increased in recent years
